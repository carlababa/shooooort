# Shooooort

The link shortener with a long name

## How to Start

Clone repository:
```
git clone https://github.com/carlababa/shooooort.git  

cd shooooort
``` 

Install Dependencies:

```
npm install
```

Build project:

```
npm run build
```

Start server:

```
npm start
```
Access [localhost:3000](http://localhost:3000)

## How to test

```
npm test
```
